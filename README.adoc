= Spring Cloud Gateway Native Demo

== Requirements

One of:

* GraalVM with native-image installed (`gu install native-image`)
* Docker

== Build & Run

.Binary build
 $ ./gradlew nativeCompile
 $ ./build/native/nativeCompile/gateway-native

.K8s deployment
 See ./scripts/build-and-deploy.sh
